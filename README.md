# CO2Monitor
CO2 Monitoring 

# Hardware
    - Arduino Mega 2560
    - K30 fast response CO2 Sensor

Read CO2 value from K30 fast response with Serial Communication 
and Display to Serial monitor every 5 second

Connection
K30 Tx  -> Mega Serial1 Rx
K30 Rx  -> Mega Serial1 Tx
K30 Vcc -> Mega 5V
K30 G   -> Mega Ground
